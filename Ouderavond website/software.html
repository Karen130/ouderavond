Software

Tijdens de opleidingen Software Development en Creative Media Technology leer je werken op basis van verschillende projecten.

Binnen een project werk je samen, maar er zijn ook projecten waarbij je volledig zelfstandig werkt. Bij het ontwerpen en bouwen leer je gebruik te maken van verschillende programmeertalen.

Bij software development zijn dit de programmeertalen HTML, PHP, JavaScript, C# en Python. Bij Creative Media Technology zijn dit HTML/CSS, Sass, Javascript, React.js en Next.js.
